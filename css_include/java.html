Introduction to JavaScript

JavaScript is a powerful, dynamic programming language primarily used in web development. It allows developers to create interactive web pages, handle events, manipulate the DOM, work with APIs, and build both client-side and server-side applications. Modern frameworks like React, Vue, and Node.js heavily rely on JavaScript.

1) What is eval in JavaScript and why can it be blocked?

eval() is a built-in function that takes a string and executes it as JavaScript code. For example: eval("2 + 2") returns 4. It can also evaluate expressions or even function definitions.

Security risk: Using eval with untrusted input can allow attackers to run arbitrary code. For this reason, modern browsers may block it via Content Security Policy (CSP), unless 'unsafe-eval' is explicitly allowed.

Eval Usage Examples
Code Result Blocked?
eval("2 + 2") 4 No
eval(userInput) Depends on input Yes if CSP blocks unsafe-eval

2) Does JavaScript short-circuit logical expressions?

Yes. Logical operators || (OR) and && (AND) short-circuit: they stop evaluating as soon as the result is known.

Example: in true || doSomething(), the second part is not executed because the OR operator already returns true. 

true || X   --> X is NOT evaluated
false || X  --> X is evaluated
true && X   --> X is evaluated
false && X  --> X is NOT evaluated

3) Can a method accept 1000 arguments without writing 1000 functions?

Yes. JavaScript functions can handle any number of arguments using the arguments object or the ES6 rest parameter syntax (...args). This eliminates the need to write many separate functions.

Handling Multiple Arguments
Method Example Notes
Rest parameter function f(...args) { } args is an array, can handle 1000+ values
Arguments object function f() { console.log(arguments); } Older method, array-like object

4) Does JavaScript support pre and post incrementation?

Yes. JavaScript supports ++i (pre-increment) and i++ (post-increment).

Pre-increment: increases the value first, then returns it.
Post-increment: returns the current value, then increases the variable.

Pre vs Post Increment
Operator Behavior Example
++i Increment first, then return i=1; b=++i; // b=2, i=2
i++ Return first, then increment i=1; a=i++; // a=1, i=2
